## 🛠️ Immediate Features: "Design Evolution Workspace"

### 1. 🗂️ Customized Export Modal
Replace the single "Download" button with a granular, developer-centric export experience.
- **Granular Selection**: Users can choose exactly which tokens to include (`[x] Colors`, `[x] Shadows`, `[x] Spacing`, etc.).
- **Format Flexibility**: Toggle between `CSS Variables`, `Tailwind Config`, and `Style Dictionary JSON`.

### 2. ✨ Ideology Remix (Color Scheme Generator)
Transform the "Math" of a design system into a new "Vibe" while maintaining professional contrast and ratios.
- **"Remix Ideology" Button**: Located in the extraction results panel.
- **Vibe Shift Engine**: AI-driven re-calculation of hues/saturation.
  - *Moods*: "Cyber-Noir," "Desert Sunset," "Arctic Minimal," "Vibrant Growth."
- **Apply/Commit Workflow**:

  - **Undo/Redo**: A persistent toast/banner to revert the remix if the result isn't perfect.
- **State Management**: Ensure all linked CSS variables in the UI are updated reactively on "Apply."

### 3. 🏷️ Semantic Token Mapping (The Pomelli Lesson)
Move beyond raw CSS values to intentional brand tokens.
- **Context Awareness**: Categorize tokens by use-case (e.g., `brand-primary-action`, `surface-background-muted`).
- **Section Detection**: Identify "Hero," "Pricing," and "CTA" zones to group tokens logically in the export.
