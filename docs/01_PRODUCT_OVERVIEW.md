# Product Overview

## Product Name (Working)
TokenExtract

## One-Line Description
Paste a landing page URL → Get clean, production-ready CSS design tokens.

## Target User
Primary: Frontend developers building landing pages and SaaS products.

## Core Problem
Developers manually inspect:
- Colors
- Fonts
- Border radius
From inspiration sites.

This takes 20–40 minutes per site.

## Solution
Automated extraction + normalization + structured CSS variable output.

## V1 Scope (Strict)

Extract:
- Top 5–8 colors (clustered + deduplicated)
- Heading font family
- Body font family
- Border radius scale (3 sizes max)

Ignore:
- Shadows
- Spacing scale
- Animations
- Gradients
- Multi-page crawling

## Success Criteria (V1)

- Extraction works on 70%+ modern landing pages
- Output is clean and minimal
- Processing time < 20 seconds
- Developers can copy & use instantly
